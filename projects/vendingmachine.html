Vending Machine

Heavy C++ Programming

Project Overview

This project involved developing a C++ front-end simulation of a vending machine for a fictional pastry company. The primary focus was on implementing robust programming techniques, clean code, and dynamic memory management while following modern C++14 standards.

Key Objectives & Skills Demonstrated

  • Core C++ Concepts: Implementation of Abstract Data Types (ADTs), linked lists (singly, doubly, and 2D), pointers, and dynamic memory management.
  • File Processing: Loading and saving inventory and coins using structured file formats.
  • Program State Management: Tracking vending machine and transaction states for correct operation.
  • Exception Handling & Validation: Handling invalid input and ensuring program stability.
  • Testing & Debugging: Creation of test cases and verification of memory safety using Valgrind.
  • C++14 and STL Usage: Leveraging modern language features and standard library utilities for efficiency.
  • Design Patterns: Implementing the Command pattern for menu operations to enhance modularity and scalability.
  • Teamwork and Collaboration: Working effectively in a group environment using Git for version control and project coordination.

The project was built entirely from scratch, emphasizing modular design, proper coding style, and adherence to software engineering principles, resulting in a fully functional and maintainable vending machine simulation.
